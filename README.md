# DSW1_T2_CONTRERASQUINTANA_WEB

## Descripción
Proyecto frontend desarrollado en HTML, CSS y JavaScript para consumir la API de la Biblioteca.  
Permite ver libros, crear nuevos registros y editar o eliminar existentes desde la interfaz web.

## Configuración
1. Asegurarse de que la API `library-api` esté corriendo.
2. Clonar o abrir este proyecto en Visual Studio Code.

## Ejecutar el proyecto
1. Abrir la carpeta del proyecto en VS Code.
2. ejecutar npm run dev ingresar con el localhost generado

## Funcionalidades
- Visualizar libros existentes.
- Crear nuevo libro.
- Editar libro.
- Eliminar libro.
- Actualización de estadísticas (número de libros, categorías, etc.).
